Step 1:

Click the use button. Now you will find the MW Typer node in the Hierarchy window and the MW_Typer script in the assets.


Step 2:

Drag the MW Typer node to under the Canvas. If there is no Canvas, please create one. Now, run the project and you will see the typer effect.


Step 3:

To speed up or slow down the typing, you may adjust the scheduleInterval in the Inspector tab or just call setScheduleInterval() in your code.

You may call go() to show the typer effect whenever you like. (To do this, remember to delete this.go(); in MW_Typer.ts)

Here is a snippet:

@property({type: Node}) MWTyperNode: Node = null! start () { let MWTyperComponent = this.MWTyperNode.getComponent(MwTyper)!; MWTyperComponent.setScheduleInterval(0.8); MWTyperComponent.go(); }
