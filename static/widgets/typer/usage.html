Step 1:

Click the use button. Now you will find the MW Typer node in the Hierarchy tab and the MW_Typer script in the assets.


Step 2:

Drag the MW Typer node to under the Canvas. If there is no Canvas, please create one. Now, run the project and you will see the typer effect.


API:

/* Show the typer effect. */ show() /* Hide the typer effect. */ hide() /* Adjust the typing speed. */ setScheduleInterval(seconds: number)
