Step 1:

Click the use button. Now you will find the MW Tooltip node in the Hierarchy tab. This node is created from the MW Tooltip prefab in the assets.


Step 2:

Drag the MW Tooltip node to under the Canvas. If there is no Canvas, please create one.


Step 3:

In your own script, you can call setTooltip() to set a tooltip for a node, and call cancelTooltip() to kill the tooltip of a node.


API:

/* Set the tooltip for a node. */ setTooltip(targetNode: Node, str: string) /* Kill the tooltip of a node. */ cancelTooltip(targetNode: Node)
