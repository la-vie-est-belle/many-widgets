Step 1:

Click the use button. Now you will find the MW ComboBox node in the Hierarchy tab. This node is created from the MW ComboBox prefab in the assets.


Step 2:

Drag the MW MovingBackground node to under the Canvas. If there is no Canvas, please create one. Now, run the project and you will see the combo box.


API:

/* Set the items in the combo box. */ setItems(stringArray: string[]) /* Get the current text shown on the combo box. */ getCurrentText() /* Set the current text shown on the combo box. Attention: you need to call setItems(array: string[]) to include this text into items. */ setCurrentText() /* Set the sign when the combo box is open. */ setOpenSign() /* Set the sign when the combo box is closed. */ setClosedSign()
