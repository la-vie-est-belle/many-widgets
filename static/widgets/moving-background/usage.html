Step 1:

Click the use button. Now you will find the MW MovingBackground node as well as its children Background 1 and Background 2 in the Hierarchy tab. A MW_MovingBackground script is created in the assets.


Step 2:

Drag the MW MovingBackground node to under the Canvas. If there is no Canvas, please create one. Now, fill the two children node with sprite frames. Run the project and you will see the moving sprites.


API:

/* Move the background. */ move() /* Stop moving the background. */ stop() /* Adjust the background move speed. */ setMoveSpeed(speed: number)
