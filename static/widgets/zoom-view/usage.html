Step 1:

Click the use button. Now you will find the MW ZoomView node and the Sprite node in the Hierarchy tab. A MW_ZoomView script is created in the assets.


Step 2:

Drag the MW MovingBackground node to under the Canvas. If there is no Canvas, please create one. Now, fill the sprite node with a sprite frame. Run the project on a devic with touch screen and you will see the zoom effect.
