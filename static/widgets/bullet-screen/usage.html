Step 1:

Click the use button. Now you will find the Bullet Screen prefab and the MW_BulletScreen script in the assets.


Step 2:

Make the MW_Bullet_Screen.ts as a component of the Canvas. If there is no Canvas, please create one. Then drag the Bullet Screen prefab onto the MWBulletScreen component.


API:

/* Set text shown on the screen. */ setBullets(textArray: string[]) /* Show the bullet screen. */ show() /* Hide the bullet screen. */ hide() /* Adjust the interval of spawning the bullets. */ setSpawnInterval(interval: number) /* Adjust the number of bullets Attention: as Math.random() is used in the source code, so the real number of bullets is actually no bigger than the number set by setBulletsNum(). */ setBulletsNum(num: number) /* filter or allow the repeat in each spawn */ setAllowRepeated(status: boolean)
