Step 1:

Click the use button. Now you will find the Child Layout prefab and the MW_RollingNumber script in the assets. You will also find the MW RollingNumber node in the Hierarchy tab.


Step 2:

Drag the Child Layout prefab onto the MWRollingNumber component.


API:

/* Set the number that will be shown after rolling. Call this method before show() as long as the number is modified. */ setNumber(numStr: string) /* Show the rolling effect. */ show() /* Set the time needed to roll. */ setRollTime(seconds: number)
