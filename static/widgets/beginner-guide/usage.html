Step 1:

Click the use button. Now you will find the MW BeginnerGuide node as well as its children background and instruction in the Hierarchy tab. A MW_BeginnerGuide script is created in the assets.


Step 2:

The MW BeginnerGuide node must be placed under the Canvas and should be its last child. Otherwise the mask won't cover some nodes. Now set the background a plain sprite frame.


Step 3:

In your own script, call the setInstructions() and the setClickNodes() API to set the instructions and the nodes to click, and at last call run(). The project is good to go.


API:

/* Set the instructions. Each click should has an instruction. */ setInstructions(instructionsArray: string []) /* Set the nodes the beginners are to click. */ setClickNodes(nodePathArray: string[]) /* Show beginner guide. */ show() /* Hide the guide. */ hide() /* Check if the guide is over. */ isGuideOver()
