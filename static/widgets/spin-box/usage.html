Step 1:

Click the use button. Now you will find the MW SpinBox node in the Hierarchy tab. This node is created from the MW SpinBox prefab in the assets.


Step 2:

Drag the MW SpinBox node to under the Canvas. If there is no Canvas, please create one. Now, run the project and you will see the spin box.


API:

/* Set the current value of the spin box. */ setValue(num: number) /* Set the maximum value of the spin box. */ setMaxValue(num: number) /* Set the minimum value of the spin box. */ setMinValue(num: number) /* Set the value step on each add or subtraction */ setSingleStep(num: number)
